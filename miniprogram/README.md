# 鲨曼 Shark Mind Club 小程序

这是将React应用转换为微信小程序的项目。

## 项目结构

```
miniprogram/
├── app.js              # 小程序入口文件
├── app.json            # 小程序配置文件
├── app.wxss            # 小程序全局样式
├── pages/              # 页面文件夹
│   ├── home/          # 首页
│   │   ├── index.js
│   │   ├── index.wxml
│   │   ├── index.wxss
│   │   └── index.json
│   ├── ranking/       # 排行榜页面
│   ├── profile/        # 个人资料页面
│   ├── order/          # 订单页面（预留）
│   └── pickup/         # 取餐页面（预留）
├── styles/             # 样式文件夹
│   └── common.wxss    # 公共样式
├── data/               # 数据文件夹
│   └── mockData.js    # 模拟数据
├── images/             # 图片资源文件夹
│   └── tabbar/        # 底部导航图标
└── README.md          # 项目说明
```

## 功能特性

### 已实现功能
- ✅ 首页：活动展示、扫码参赛、点餐、各种指引
- ✅ 排行榜：周榜、月榜、季榜、年榜、进圈率榜单
- ✅ 个人资料：用户信息、评价设置、登录功能
- ✅ 底部导航：页面切换
- ✅ 全局状态管理：用户状态、购物车、订单
- ✅ 响应式设计：适配不同屏幕尺寸

### 主要组件
- 活动模态框
- 排行榜列表
- 玩家评价系统
- 登录设置界面
- Toast提示系统

## 开发说明

### 环境要求
- 微信开发者工具
- 支持ES6语法

### 如何运行
1. 使用微信开发者工具打开 `miniprogram` 文件夹
2. 点击"编译"运行小程序

### 数据说明
- 使用 `data/mockData.js` 中的模拟数据
- 全局状态通过 `app.js` 管理
- 页面间数据通过全局状态同步

## 注意事项

### 图片资源
目前使用网络图片资源，在生产环境中需要：
1. 将图片下载到本地 `images/` 文件夹
2. 更新代码中的图片路径
3. 考虑图片压缩和优化

### API适配
小程序版本使用模拟API，实际部署时需要：
1. 替换为真实的API调用
2. 添加错误处理
3. 实现数据缓存策略

### 样式适配
- 使用 `rpx` 单位适配不同屏幕
- 颜色和间距参考设计稿调整
- 测试不同机型的显示效果

## 技术栈

- **框架**: 微信小程序原生框架
- **语言**: JavaScript ES6
- **样式**: WXSS (类似CSS)
- **数据**: JSON + JavaScript对象
- **状态管理**: 小程序全局数据 + 页面数据

## 转换说明

本项目由React应用转换而来，主要变化：

### React → 小程序
- `JSX` → `WXML` 模板语法
- `useState` → `this.setData()`
- `props` → `data-` 属性传递
- `Tailwind CSS` → `WXSS` 样式
- `React Router` → 小程序页面路由

### 组件适配
- Radix UI组件 → 原生小程序组件
- 自定义Hooks → 小程序生命周期方法
- Event handlers → 事件绑定函数
